## 在终端中使用sublime

### alias命令配置别名
1. 打开用户配置文件
```
vim ~/.bash_profile
```
2. 添加下面配置
```
alias subl="'/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl'"
```
3. 保存后执行下命令使其生效
```
source ~/.bash_profile
```

### 终端命令行sublime使用
```
subl ~/.bash_profile
```
